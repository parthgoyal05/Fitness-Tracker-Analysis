# 📌 Overview

The **Strava Fitness Data Analysis Project** explores user activity data generated from smart fitness devices to understand behavioral patterns related to **physical activity, calorie expenditure, and sleep habits**.

Modern wearable devices collect large volumes of health-related data such as **steps taken, calories burned, sleep duration, and activity intensity**. By analyzing this data, we can uncover patterns that help improve **health tracking systems, user engagement, and wellness product strategies**.

This project performs **Exploratory Data Analysis (EDA)** using:

- **SQL (SQLite)** for structured querying and data cleaning  
- **Python (Pandas, Matplotlib, Seaborn)** for data analysis and visualization  
- **Tableau** for interactive dashboard creation  

The goal is to extract meaningful insights that can help **fitness technology companies understand user behavior and improve their products.**

---

# 🎯 Problem Statement

Multiple datasets containing smart fitness device data are available, including:

- Daily physical activity
- Steps taken
- Calories burned
- Sleep records
- Activity intensity

The challenge is to **clean, merge, and analyze these datasets** to understand how users interact with fitness tracking devices.

Through this analysis, we aim to identify:

- Activity patterns
- Sleep trends
- User engagement with wearable fitness devices

---

# 🎯 Business Objectives

The main objectives of this analysis are:

1. Analyze **daily activity levels** such as steps, distance, and calories burned.
2. Identify patterns in **sleep behavior and fitness routines**.
3. Understand **user engagement with wearable devices**.
4. Generate insights that can support **data-driven product and marketing strategies**.
5. Visualize trends using **Python data visualization libraries**.

---

# 🔑 Key Insights

The analysis revealed several interesting trends:
	•	Many users record less than the recommended 10,000 daily steps.
	•	Higher step counts are strongly associated with higher calorie expenditure.
	•	Most users fall into the moderately active category.
	•	Users spend a significant amount of time in sedentary activity.
	•	Most users sleep between 6–7 hours per night.

# 💡 Business Insights

From a business perspective, the analysis suggests:
	•	Increasing user step goals can improve engagement.
	•	Sleep tracking features can enhance health awareness.
	•	Activity insights can enable personalized health recommendations.
	•	Fitness device companies can leverage data insights for targeted marketing strategies

# 🏁 Conclusion

This project demonstrates how data analytics can extract meaningful insights from wearable fitness device data.

By combining SQL, Python, and Tableau, the project successfully analyzed patterns in:
	•	User physical activity
	•	Sleep behavior
	•	Calorie expenditure

These insights can help wellness technology companies improve product engagement and promote healthier lifestyles.
